機場接送

海外志工報到調查表：[連結]

抵達臺灣

  1. 我們機場接送的時間為臺灣時間上午9點(09:00)至下午5點(17:00),謝謝。
  2. 如果您於7月4日已抵達臺北，但尚未找到當日住宿地點，我們已在劍潭海外青年活動中心預留了房間(但是房間數量有限)，我們將以先抵達先安排的原則進行分配，當天所產生的額外接機及住宿費用(使用者付費須自行負擔-可以與同住或先行抵達須需接機志工共同分攤)。
    如果志工有須提前接機的需求，請致電陳先生(臺灣手機號碼) 0935-261-165。
  3. 如果您的班機抵達時間晚於臺灣時間7月6日晚上5點(17:00) ，請在「海外志工報到調查表」載明您接機需求時間，我們將另作安排。

送機

  1. 送機時間自7月30日下午2點(14:00)起至8月1日晚上6點(18:00)，由於人數眾多量能有限，我們將以海外志工為優先安排送機，恕無另受理海外志工家屬及朋友預訂送機安排服務。
  2. 活動閉幕(結業)式，將於7月30日中午12點(12:00)前結束，親友可在閉幕(結業)式後或8月1日中午12點(12:00)前來劍潭青年活動中心接回海外志工，謝謝。
  3. 如果海外志工離臺的班機時間在8月1日中午之後，當天的餐飲(晚餐)需由海外志工自理。
  4. 如果海外志工離臺的班機時間在8月2日之後(或者更晚)，我們將協助安排住宿及餐飲及送機交通，相關費用(使用者付費須自行負擔-由我們安排與同性別志工同寢室)。

飲食習慣相關問題

  1. 在教育訓練週(7月4日至10日)期間，主要安排於劍潭海外青年活動中心餐廳(桌餐方式)用餐，如有對特定食材過敏或特殊飲食(如豬肉、雞肉等)需求敬請提前告知，我們將提供素食與葷食兩種選擇另外安排。
  2. 在校服務週(7月11至24日)飲食由服務學校統一安排。
  3. 在寶島參訪週(7月25至29日)飲食安排將會在旅遊行程中安排在地特色美食(特殊飲食另外安排)。
  4. 活動自7月4日報到當天晚餐起至7月30日閉幕(結業)式晚餐止，期間我們將提供每日三餐(在校服務週亦會由學校師長安排)。

服裝

  1. 我們會贈送每位志工2件團體制服(上衣)，於7月6日報到時依志工報名表所填尺寸領取，如有尺寸不合只能在所有志工報到後與其他志工或不會來報到的志工交換。
  2. 活動時間自7月4日至7月30日，營期時間長，且第一週教育訓練結束後及第三週和寶島旅遊週都必須志工自行搬運行李，攜帶太多物品只會讓志工更辛苦，建議攜帶4-5套換洗衣服即可。
  3. 第一週教育訓練週6天都須穿著團體制服，第二、三週則依學校規定(往年也是14天都須穿團體制服)，第四週寶島旅遊我們會開放3天穿著自己的便服，可以開放穿短褲，其他時段請配合穿著長褲或7分褲。住宿飯店有投幣式洗衣機和烘衣機，也有免費脫水機，但數量有限，我們會贈送洗衣袋，建議同寢室室友一起洗衣服，或自己手洗、脫水後晾乾即可。基本上洗衣房不會關閉，但因為志工人數實在太多，為了避免排隊整晚洗衣洗到天亮，建議志工每天晚上就寢前可以利用洗澡的時間順便洗衣服，團體衣服都是吸濕排汗材質，晾在房間吹冷氣，通常可在隔日早晨變乾。
  4. 如果不方便攜帶洗衣精，在劍潭中心有小包裝洗衣精可以購買，到學校服務學校會準備。
  5. 依飯店不同房間內約有4-6個衣架，建議可自行攜帶2-3個衣架晾衣服。
  6. 請您幫忙配合穿著長褲，除了學校教學外，還會有始業式或結業式較正式的場合，穿著短褲真的不太合適，下半身盡量以薄長褲、長運動褲為主，短褲、緊身牛仔短褲、韻律褲或運動短褲較不適合。

住宿

  1. 教育訓練週和寶島參訪週原則上志工4-6人一間房，我們有準備盥洗包(小毛巾、牙膏、牙刷、香皂、梳子、外袋)每人一份，飯店房間也有沐浴乳、洗髮精及吹風機。
  2. 學校教學兩週住宿依學校安排各校不盡相同，有的學校住在學校裡面，有的住飯店、民宿或學生家長家裡(home stay)，但不管住哪裡，學校有老師或警衛在，飯店和民宿也有服務人員，老師們都會留電話給志工，老師們和我的手機都是24小時不關機的。臺灣暑假酷熱，除了高山地區少用冷氣外，其他地區的學校教室和住宿地點幾乎都有冷氣。
  3. 盥洗包的物品(如牙膏、毛巾、香皂等)都足夠使用1個月，學校也有經費可以提供沐浴乳和洗髮精等生活用品。
  4. 第一週在劍潭中心有提供網路使用，第2.3週就要視每個學校的狀況，通常學校都會有網路(至少會有有線網路)，第4週寶島參訪旅遊只有飯店才有提供網路。
  5. 如果真的需要申辦網路卡，我們已經幫志工接洽中華電信公司，該公司會特別為本次海外志工派人到劍潭活動中心開設申辦網卡櫃臺，並推出「海外青年專案」15天30G免費網卡，30G用完降速為5mb，可持續儲值使用，卡片金額用完即可取消，因為是為海外青年專案，需準備護照正本(中華電信需掃描護照正本)才能申辦，恕不接受臺灣身分證或健保卡辦理。此項服務為免費提供，因志工人數眾多，請於「海外志工報到調查表」中華電信調查連結中先行填具，以便於7月6日報到當天領取作業。
    • 因開通需要確認入出境紀錄，因此若您申請本專案，請在離開僑居國以及入境中華民國(臺灣)時，務必使用僑居國護照。
    • 注意：僑民役男不適用本專案，詳情請參閱僑民役男相關說明。

交通

  1. 除了7月4日及7月31日提供接/送機交通接駁服務外，7月10日教育訓練週結束後學校會安排交通接送至各校，北部地區學校會派車接送，中南部地區大部分同縣市學校會共乘大巴士，更遠的會搭乘臺灣高鐵。離島的學校已經替服務志工準備好志工的機票。7月27日學校會再將志工送至寶島旅遊集合地點，搭乘大巴士參加旅遊參訪活動，活動期間皆由主辦單位提供交通接送。
  2. 在學校教學服務前間的週末假日，學校老師會安排旅遊或城市探索活動，其他週一到週五教學結束後，志工應該會忙著準備隔天的教案和課程，沒有時間可以離開學校探索城市，也為了志工的安全，避免離開學校，謝謝。
  3. 活動期間7月8日至8月1日謝絕在臺親朋好友探視志工，因為活動行程滿滿，教育訓練週到晚上9點皆有課程，教學服務兩週週一至周五都要準備隔日教案和課程，週末學校皆有安排小旅行活動，依據往年的經驗，比較有經驗(來過1-2次)的志工，晚上備課通常最早晚上8點結束，第一次參加的志工，有時候會備課到晚上10-11點，如果親友晚上到學校探視，恐怕更會耽誤志工備課和休息的時間。
  4. 7月30日結業式因為志工人數眾多，還有學校師長要參加，座位區有限，不建議邀請眾多親朋好友前來觀禮，謝謝。

僑民役男

  1. 僑民役男定義：年滿19歲~36歲之間，並同時持有中華民國(臺灣)護照與僑居國護照，以及在中華民國(臺灣)持有戶籍的男性。(簡單判定根據：請確認您的中華民國(臺灣)護照上是否有「身分證字號」欄位，若有，即代表您持有臺灣戶籍。)
  2. 若您持有僑民役男身分，請使用中華民國(臺灣)護照入境與出境。(由於中華電信提供的海外青年網路專案須使用僑居國護照出入境，因此僑民役男不適用該網路專案)
  3. 若您本次入境前未申請過出境許可與出境核准，出境前須重新申請：
    1. 向僑委會申請役政用華僑身分證明書
      1. 需填寫華僑身分證明申請書，若委託他人申請，則須加填委託書(委託人兩份都需親自簽名)
      2. 備妥中華民國與居留國護照(不能用影本)，申請一次須工本費$200元新台幣
      3. 至僑委會臨櫃申請(行政院中央聯合辦公大樓南棟(台北市中正區徐州路5號)3F)，等待半天至一天工作日即可領取。
    2. 至內政部移民署僑民役男申請出境文件審查平臺申請出境許可
      1. 審查平臺連結(開啟新視窗)
      2. 登入需填寫役男身份證字號、出生日期及中華民國(臺灣)護照號碼
      3. 登入後選擇新增申請並填寫必要資訊：
        1. 僑民役男姓名
        2. 僑民役男E-Mail(需填寫兩次，一次為確認)
        3. 僑民役男役政用華僑身分證明書(除須上傳可清楚辨識的照片外，亦須填寫有效期限)
        4. 僑民役男中華民國護照及僑居國護照(除填寫編號外，亦須上傳可清楚辨識照片及填寫有效期限)
        5. 僑民役男4個月內自僑居國出國的紀錄(最佳證明為自僑居地起飛的機票，且機票資訊必須要能清楚確認是僑民本人持有)
        6. 申請後等待約一至兩個工作天後，可再次登入查詢申請結果(連絡電話：內政部役政司，02-85013928)
        7. 若僑民役男尚未建立兵籍資料，則需親自(視情況須由監護人陪伴)前往戶籍所在地戶政事務所辦理，完成後才能獲得出境許可
    3. 至內政部移民署各地服務站臨櫃申請出境核准
      1. 須至服務站填寫申請書，若委託他人代理則須加填委託書(皆須本人親自簽名)
    4. 僑民役男在通過出境核准後，於有效期限內可自由出入境中華民國自由地區(臺灣地區)。
  4. 承辦單位可協助辦理申請，但請於活動開始前向承辦單位提出需求，活動正式開始後概不受理。

其他問題

  1. 營隊規則簽名表格可以報到當天繳交，也可以先簽名後拍照上傳至雲端連結：[連結]
  2. 如怕護照證件遺失，可事先用手機拍照存檔或是影印紙本，萬一需要就醫時可以使用影本或臺灣健保卡，但如需辦理網卡(如有家長陪同可以辦完網卡後由家長收回)
  3. 如果上飛機允許，可以攜帶些許零食，劍潭中心設有(泡麵、飲料及零食)的自動販賣機，中心服務檯亦有提供兌換零錢的服務。
  4. 除非需要大量購買臺灣名產，否則零用錢建議攜帶新臺幣1,000-2,000元即可，或是悠遊卡儲值1,000元，現金1,000元亦可。
  5. 因為報到後的晚上有些注意事項還要跟志工們說明，原則上不希望志工報到後離開劍潭中心，報到後我們有責任維護每位志工和工作人員的安全，但會安排其中一天晚上由工作人員帶志工出去逛士林夜市或西門町。
    • 其實我們並不是要限制志工的行動或說是"管制"志工，畢竟志工來臺灣人生地不熟，加上我們有責任維護參加這個活動的每個人的安全，雖然臺灣的治安算很好，但不怕一萬只怕萬一，所以麻煩各位家長，協助主辦單位(僑務委員會)跟志工們說明，營隊期間的規則和要求都是為了活動順利和大家的安全，沒有工作人員帶領下請勿私自離開營隊活動區域，也切勿飲酒，就算年滿18歲也一樣(因為團體內還有很多未滿17歲的志工，擔心會有渲染效應...難以管理)，志工來臺是為了服務...為了增加生活經驗和學習，希望各位家長和志工能夠諒解，謝謝。
  6. 是否配戴口罩相關規定：現在臺灣新冠疫情已經降級、全面解除管制!大眾運輸及公共場所無強制配戴口罩，若有個人因素考量，亦可以自行選擇是否配戴口罩，謝謝。
